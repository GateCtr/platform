# GateCtr Positioning

## Category

**AI Cost Infrastructure** — not "LLM Gateway" (too generic), not "AI Platform" (too vague).

This positions GateCtr as infrastructure whose sole job is to reduce what you pay LLM providers.

## Core Positioning Statement

> GateCtr is the layer between your code and your LLM providers that reduces your token usage by up to 40% — automatically, without touching your code.

## Primary USP

The combination of three capabilities in one SaaS, no token markup, no lock-in:
1. Automatic prompt compression (up to 40% fewer tokens sent to the LLM)
2. Cost-aware model routing (cheapest model that gets the job done)
3. Hard budget enforcement (blocks before the LLM call, not after)

## What GateCtr Is

- AI Cost Infrastructure — a layer that reduces your LLM bill automatically
- A drop-in endpoint swap — no SDK lock-in, no architecture changes
- Infrastructure for cost control, observability, and intelligent routing

## What GateCtr Is NOT

Never position GateCtr as:

- A RAG / retrieval system
- A prompt engineering tool
- A chatbot or agent builder
- A model fine-tuning platform
- A replacement for your LLM provider

If asked "is it like X?", the answer is: "GateCtr controls the traffic. X builds the app."

## Messaging Hierarchy

1. Cost reduction — "Up to 40% fewer tokens. No surprise invoices."
2. Zero friction — "One endpoint swap. No code changes."
3. Full visibility — "Every request. Every token. Measured."
4. Multi-LLM — "OpenAI, Anthropic, Mistral, Gemini. One gateway."

## Audience

Primary: Solo devs and startups paying $50–200/mo in LLM tokens who don't have time to configure complex gateways.
Secondary: Engineering teams and CTOs managing AI infrastructure costs at scale.

They are technical. They are skeptical of marketing. Lead with numbers and facts.

## Tone in Competitive Contexts

- Never disparage competitors by name in UI copy
- Use capability comparisons, not brand comparisons
- "GateCtr doesn't just log — it acts." (not "unlike Helicone")
- Never claim to be "the only" tool doing X — the market moves fast
- Defensible claim: "the only gateway combining compression + routing + hard caps in a SaaS starting at $0 with zero token markup"
